 Project Title

Simple E-Learning Portal

Project Description

The Simple E-Learning Portal is a web-based application designed to provide an easy platform for online learning. It allows students to view available courses and access learning content online. The admin can add and manage course details through a simple interface. The backend is implemented using Node.js to handle requests and data operations efficiently.

View available courses
